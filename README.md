# 天气查询快捷指令

> **冬天来了,快查查天气,以防感冒~**

## 实现思路

查询某地的天气 -> API获取 -> 经度和纬度

天气查询接口 -> 拼接经度和维度参数

> 两种查询天气方式:
>
> > 获取当前天气:
> >
> > 将天气查询API 参数num替换为10 ->  请求API -> 通过词典获取对应的天气信息
> >
> > > **获取天气预报信息**
> > >
> > > 由于不同地区天气预报信息长度不一样，所以需要判断
> > >
> > > 但是长度2是固定的,但是3是不固定的。
> > >
> > > 判断当长度 > 2时 获取 数组3当中的元素值.
>
> > 获取未来天气:
> >
> > 请求输入要查询的天气天数 -> 将天气查询API 参数num替换为输入要查询的天气天数 ->  请求API -> 通过词典获取对应的天气信息
